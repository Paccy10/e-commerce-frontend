# E-commerce

## Installation and Setup

- Clone the repository

```
https://github.com/Paccy10/e-commerce-frontend.git
```

- Install dependencies

```
npm install
```

- Run the application

```
npm start
```

- Running tests and generating report

```
npm test
```
